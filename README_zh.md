# 黑曼陀罗

不可预知的黑暗，死亡和颠沛流离的爱，凡间的无爱与无仇，绝望的爱，不可预知的死亡和爱，被伤害的坚韧创痍的心灵，生的不归之路

## 介绍

[zap][1] 和 [lumberjack][2] 的二次封装，集成了 [Gin][3] 的 logging 中间件


## 使用

```go
import (
	log "github.com/FlowerLab/blackdatura"
)
```

查看 [示例](example/main.go)
